- إكمال إدارة الطلبات.
- تحسين نظام التوصيل.
- تحسين Dashboard والإحصائيات.
- إصلاح أي Validation أو Type Errors متبقية.
- التأكد أن جميع العمليات تعمل فعلياً وليست UI فقط.

المطلوب بالتفصيل
━━━━━━━━━━━━━━

1) المنتجات والمتجر
- تأكد أن:
  - إضافة منتج تعمل.
  - تعديل منتج يعمل.
  - حذف منتج يعمل.
  - رفع الصور يعمل.
  - حذف الصور يعمل.
  - اختيار الألوان يعمل.
  - الكمية تعمل (+ / -).
  - الوصف يظهر بشكل مرتب.
  - الصور تظهر كاملة بدون قص.
  - Preview للصورة يعمل.

2) الأقسام والخدمات
- إدارة الأقسام الرئيسية والفرعية تعمل بالكامل.
- رفع صورة للقسم يعمل.
- حذف صورة القسم يعمل.
- ترتيب الأقسام يعمل.
- إخفاء/إظهار القسم يعمل.

3) السلة وطلبات المتجر
- السلة تحدث مباشرة بدون Refresh.
- تعديل الكمية داخل السلة يعمل.
- حذف المنتج من السلة يعمل.
- حساب المجموع صحيح.
- التغليف يضاف تلقائياً.
- التوصيل يضاف تلقائياً.
- إنشاء الطلب يحفظ كل البيانات.

4) التوصيل والموقع
- تحديد الموقع يعمل.
- Google Maps URL يتولد تلقائياً.
- يظهر للزبون وللإدارة.
- إذا رفض الموقع يظهر إدخال يدوي للرابط.

5) Dashboard والإحصائيات
أضف:
- إجمالي الطلبات
- إجمالي المبيعات
- الطلبات الجديدة
- الطلبات المكتملة
- أكثر المنتجات طلباً
- أحدث الطلبات

مع تحديث مباشر وواجهة مرتبة.

6) الحماية والجودة
- لا توجد Console Errors.
- لا توجد Runtime Errors.
- لا توجد Validation Errors.
- لا توجد any casts غير ضرورية.
- أصلح أي type mismatch.

7) الاختبارات النهائية
بعد الانتهاء شغّل:
npm run typecheck
npm run build

وتأكد:
- كل شيء يمر بنجاح.
- لا توجد failed validations.
- لا توجد صفحات مكسورة.
