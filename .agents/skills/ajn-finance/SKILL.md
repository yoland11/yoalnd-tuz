# AJN Finance Skill

Project Stack:

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Drizzle ORM
- PostgreSQL (Supabase)
- Supabase Storage
- Vercel
- PWA + Push Notifications

Rules:

- Preserve AJN design system.
- Do not break existing routes.
- Do not break existing APIs.
- Use safe Drizzle migrations only.
- Use RTL Arabic UI.
- Follow existing project architecture.
- Use Zod validation.
- Run pnpm run typecheck.
- Run pnpm run build.
- Fix all errors before completion.
- Do not deploy or push unless requested.
